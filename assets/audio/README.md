# 网页音频资源

`deep-space-original.mp3` 是当前网页使用的本地循环背景音乐。网页保持默认静音；访客点击顶部“声音”后，音频才会启动。

播放器会将音乐接入 Web Audio 的空间化、低通和动态限幅链路，以维持较有包围感的响度并避免削波。选择天体和镜头飞行的提示音仍由浏览器实时合成。音乐保留前 6 分钟并循环播放，文件约 14 MB。PWA 不会在首次安装时预下载该音乐；访客首次开启声音后会将其缓存，之后可离线重播。

替换背景音乐时请保留文件名，或同步更新 `src/systems/spatialAudio.js` 中的 `ORIGINAL_TRACK_URL`。只可使用原创或已获得公开网页发布授权的音频；对应的权利信息必须登记在 `docs/AUDIO_LICENSES.md`。
